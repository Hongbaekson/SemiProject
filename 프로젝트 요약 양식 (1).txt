+ 프로젝트 요약

팀명 : 숙소예약사이트 일루와 


프로젝트 제목(50자 이하) : 숙소예약과 동시에 근처맛집을 검색을 할 수 있는  사이트입니다


개요 : 
숙소정보를 한 눈에 파악하고 사용자들끼리 의견공유를 통해 숙소끼리 비교가 가능해 원하는 숙소를 빠르고 간편하게 예약할 수 있다. 


구현기능 : 
1. 숙소예약 
	- 원하는 날짜에 지역을 검색하여 숙소를 예약할 수 있도록 함
2. 숙소 북마크기능
	- 원하는 숙소를 북마크로 저장해놓을 수 있도록 함
3. 호스트등록시스템을 이용한 숙소등록
4. 댓글 기능, 후기등을 남김으로서 의견공유가 활발하게 이루어지도록함
5. 맛집추천 
6. 관리자페이지 ) 예약관리, 사용자관리 및 글관리 


설계의 주안점 : 
숙소평점을 통해 사용자에게 객관적인 정보제공
많은 숙소들의 정보를 비교하거나 실시간 인기순으로 제공하여 정보의 접근성을 높임  
근처맛집검색 제공
호스트 등록 시스템을 이용한  숙소제공

사용기술 및 개발환경 : 

	운영체제 : 	Windows 10, Mac
	서버 : 		Tomcat 9.0
	DBMS :		Oracle 11g
	개발언어 : 	JAVA, HTML, CSS, JavaScript, SQL
	개발 툴 : 		Eclipse, SQL Developer
